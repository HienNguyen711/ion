Welcome to MEET UP

The Meetup App is a mobile app for users and organisations to hold events.

What is Meetup App?

Meetup app is a social networking app specialized for meetings, events, conferences, workshops, etc

What is Meetup app Platform?

The Meetup app Platform User can create account to register for all of the events is about to happen in the world and follow their favorite organisations.

What is Meetup app Platform?

The Meetup app Platform organisation can create account to create new events, meetings, meetups, conferences, workshops for user to register
